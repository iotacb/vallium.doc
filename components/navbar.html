Navbar

The navbar component is a simple navbar component that can be used to navigate between pages.

View source code here.

Basic usage

import { Navbar } from "vallium";

function App() {
	return <Navbar>Button</Navbar>;
}

Example

import { Navbar } from "vallium";

function App() {
	return (
		<>
			<Navbar>
				<h1>
					Vallium
				</h1>
				<div>
					<a href="#">Home</a>
					<a href="#">About</a>
					<a href="#">Contact</a>
				</div>
			</Navbar>
		</>
	);
}
